# 4UP Cloud

Best files cloud

## Run Locally

Clone the project

```bash
git clone https://github.com/Ameriq8/4up-cloud.git
```

Go to the project directory

```bash
cd 4up-cloud
```

Install dependencies

```bash
npm install
# or
yarn install
```

Start the server

```bash
npm start
# or
yarn start
```
